# DendroDocs Community Health Files

This repository contains community health files and templates for the DendroDocs organization.
These files help maintain consistency and quality across all DendroDocs projects.

## Files Included
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `ISSUE_TEMPLATE/`
- `PULL_REQUEST_TEMPLATE.md`
- `SECURITY.md`

## Purpose
By centralizing these files, we ensure that all projects within the DendroDocs organization follow the same guidelines and standards, fostering a welcoming and productive community.
